4%法則 Link to heading

1994年美國財務顧問威廉‧ 班根（William P. Bengen）研究，在股債配置各50%，通膨約3%前提假設。 Link to heading

每年從退休帳戶的投資組合中領出4%作為日常生活費資金可持續領30年以上。 Link to heading

退休金＝預計每年生活費÷4％。 Link to heading

50/30/20預算法則 Link to heading

每個月50%用在必要日常花費，30%用在想提升生活品質花費，20%儲蓄理財。 Link to heading

增加被動收入 Link to heading

非工資收入，在睡覺時也能賺錢。 Link to heading

被動收入7種方式 Link to heading

  1. 當賺錢公司的股東。
  2. 取得房地產當房東收租。
  3. 經營一個高流量數位媒體(Youtobe流量收入、Podcast)。
  4. 出租設備或工具。
  5. 線上課程(智識衛星SAT、Hahow)。
  6. 投資孳息(股票配股配息或是基金股利)。
  7. 創作者授權金與智慧財產（IP）衍生收入。
